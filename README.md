# 星幕私人影吧运营系统

一款轻量级的私人影吧日常运营管理系统，包含包间管理、影片库、预约排期、到店核验、消费结算等完整功能。

## ✨ 功能特性

### 🏠 包间状态管理
- 实时查看9个包间的使用状态（空闲/使用中/维护中）
- 包间类型：情侣小包、商务中包、豪华大包、VIP派对包
- 展示当前预约信息、容纳人数、小时价格
- 点击查看包间详情和预约排期

### 🎞️ 影片分类库
- 8大影片分类（动作、爱情、科幻、喜剧、恐怖、动画、纪录片、经典）
- 内置23部经典影片数据
- 支持按分类筛选和关键词搜索
- 可手动添加新影片

### 📅 预约排期
- 快速新建预约：选择包间、影片、顾客信息、时间段
- 自动计算预估费用（小时单价 × 使用时长）
- 预约列表：状态/日期筛选、查看核验码、取消预约
- 自动检测时间冲突，避免重复预约

### ✅ 到店核验
- 输入6位核验码快速核验入场
- 展示2小时内待入场预约列表，一键核验
- 支持提前30分钟入场的时间校验
- 核验成功自动更新包间状态为"使用中"

### 💰 消费结算
- 自动计算实际观影时长
- 支持额外收费（小吃、饮料等）
- 多种支付方式：现金/微信/支付宝/银行卡/会员卡
- 实时计算折扣优惠和最终应付金额
- 结算完成自动释放包间为"空闲"

### 📊 运营总览
- 包间总数/空闲/使用中 实时统计
- 今日预约数、营收金额
- 即将开始的预约提醒
- 支付交易记录查询

## 🚀 快速启动

### 方式一：双击启动（推荐）
直接双击 **`启动系统.bat`**，脚本会自动：
1. 安装项目依赖（首次启动时）
2. 初始化数据库（首次启动时）
3. 启动服务

### 方式二：命令行启动
```bash
# 1. 安装依赖
npm install

# 2. 初始化数据库（仅首次）
npm run init-db

# 3. 启动服务
npm start
```

### 访问系统
启动成功后，在浏览器打开：
- 本机访问：**http://localhost:3000**
- 局域网访问：**http://你的电脑IP:3000**

## 🔧 其他脚本

| 脚本文件 | 功能 |
|---------|------|

## 📁 项目结构

```
.
├── server.js              # 后端服务（Express API）
├── init-db.js             # 数据库初始化脚本
├── db.js                  # 数据库连接模块
├── package.json           # 项目配置
├── 启动系统.bat           # Windows一键启动脚本
├── 重置数据库.bat         # 数据库重置脚本
├── public/
│   ├── index.html         # 前端页面
│   ├── style.css          # 样式文件
│   └── app.js             # 前端交互逻辑
└── cinema.db              # SQLite数据库文件（自动生成）
```

## 💡 使用流程建议

1. **预约** → 前台或电话预约，创建订单获得核验码
2. **到店** → 顾客到店出示核验码，在"到店核验"页面验证
3. **观影** → 核验成功，包间状态自动切换为"使用中"
4. **结算** → 观影结束，在"消费结算"页面完成付款
5. **完成** → 结算成功，包间自动释放为空闲状态

## 📝 注意事项

- 系统默认监听 **3000 端口**，可通过环境变量 `PORT` 修改
- 数据库文件 `cinema.db` 自动生成，请勿随意删除
- 如需重置数据，运行 `重置数据库.bat`
- 建议使用 Chrome / Edge / Firefox 等现代浏览器访问
